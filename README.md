# Мой грузовой бизнес — тестовая Android-версия

Минимальная версия: Android 9. Целевая версия: Android 15. Проверочный телефон: POCO X3 Pro, Android 12+.

## Реализовано

- интерфейс нового заказа, истории, клиентов, аналитики и маршрута;
- локальное хранение рабочих данных;
- ключи DaData и openrouteservice шифруются через Android Keystore;
- проверка подключений;
- получение координат российских адресов через DaData;
- оптимизация порядка точек через openrouteservice;
- открытие текущего адреса системной `geo:`-ссылкой;
- телефонные звонки через `tel:`;
- отметка выполненных точек.

## Безопасность

Ключи не вшиты в APK и не входят в репозиторий. Они вводятся пользователем в настройках и сохраняются в зашифрованном виде на устройстве.

## Сборка через GitHub Actions

1. Загрузить содержимое папки в приватный репозиторий GitHub.
2. Открыть вкладку Actions.
3. Запустить workflow `Build test APK`.
4. Скачать артефакт `my-cargo-business-test-apk`.

Локальная сборка: открыть проект в Android Studio или выполнить `gradle assembleDebug` при установленном Android SDK 35.
